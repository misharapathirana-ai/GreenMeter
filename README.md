# GreenMeter
A Utility Management System 

A database-driven solution for Sri Lankan electricity, water, and gas providers.

## Structure
- `/web` – PHP frontend (runs on XAMPP/Apache)
- `/sql` – Full MySQL database dump (`ums.sql`)

## How to Run
1. Import `sql/ums.sql` into phpMyAdmin (create DB `ums`)
2. Copy `web/` to `htdocs/ums/`
3. Visit `http://localhost/ums/`
